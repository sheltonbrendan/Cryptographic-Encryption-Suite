How AES & Symmetric Ciphers Work

  • Shared Secret Key: A single cryptographically strong key handles both encryption and decryption routines. If key security is breached, the data is completely compromised.
  • Fernet Implementation: This backend uses Python's Fernet specification, driving 128-bit AES block cipher operations configured in Cipher Block Chaining (CBC) mode under the hood.
  • Cryptographic Integrity: Fernet automatically bonds an initialization vector (IV) and an HMAC (Hash-based Message Authentication Code) signature to the raw token. This prevents padding-oracle vulnerabilities and drops altered, tampered payloads during execution.
  • Stateless Lifecycle: Because standard HTTP requests do not preserve parameters across sessions, a secure, unique sequence runtime token is independently generated inside each separate submission envelope.
