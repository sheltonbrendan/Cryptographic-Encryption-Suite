// How Caesar Cipher Works

  • Each letter is shifted forward by a fixed number in the alphabet
  • With shift 3: A → D, B → E, Z → C (wraps around)
  • Only 25 possible keys — easily broken by brute force
  • Used by Julius Caesar to protect military communications (~58 BC)

// How Vigenère Cipher Works

  • Uses a repeating keyword instead of a single shift value
  • Each letter of the keyword determines the shift for that position
  • Example: key "KEY" applies shifts of 10, 4, 24 repeating across the message
  • Resisted cryptanalysis for 300 years before being broken by Kasiski in 1863
