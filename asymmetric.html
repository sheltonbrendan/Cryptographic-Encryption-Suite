Asymmetric Encryption (RSA)

Asymmetric encryption uses a mathematically linked key pair — a public key to encrypt and a private key to decrypt. The two keys are different, yet connected.

// How It Works

  • A 2048-bit RSA key pair is generated on each request
  • The public key encrypts the message using OAEP padding
  • Only the matching private key can decrypt it
  • Used in HTTPS, email encryption, and digital signatures
  • Much slower than symmetric encryption — typically used to exchange keys
